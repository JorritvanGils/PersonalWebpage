BOX21 Computer Vision Research Engineer

Since November 2011 I started working as developer for computer vision platform BOX21 with guidance of Drs. R. Hollands. BOX21 is a Graphical User Interface that allows users to automatically extract object information from images by applying localization, classification or pose estimation without the need to program. This way BOX21 aims to make computer vision techniques accessible to the wider public.

What has increased my skills as a programmer most was understanding Object Oriented Programming and facilitation of the communication between Python and the database via SQLAlchemy. In addition, I also learned to work in a command-line interface, building unit tests, debugging and understanding how to collaborate on GitHub projects.

The BOX21 system runs via Docker containers and is programmed in programming language Python. By taking on increasingly larger tasks this job offered me the opportunity to not only understand but also apply and modify code of Neural Networks systems. As the platform is expected to grow, a task yet to be accomplished is to implement several other NN in addition to YOLOv5 so that the users can choose the best fit for their dataset.
